NAME

WeBWorK::ContentGenerator::Instructor::ProblemSetList - Entry point for Set-specific data editing/viewing

What do we want to be able to do here?

filter sort edit publish import create delete

Filter what sets are shown: - none, all, selected - matching set_id, visible to students, hidden from students

Sort sets by: - set name - open date - due date - answer date - header files - visibility to students

Switch from view mode to edit mode: - showing all sets - showing listed sets - showing selected sets Switch from edit mode to view and save changes Switch from edit mode to view and abandon changes

Make sets visible to students (publish) or hidden from students (unpublish): - none, all, listed, selected

Import sets: - single or multiple - with set name (only for single) - assign to: - only the current user - all users

Export sets: - all, listed, selected

Score sets: - none, all, selected

Create a set with a given name - as new empty set or as duplicate of first selected

Delete sets: - none, selected
