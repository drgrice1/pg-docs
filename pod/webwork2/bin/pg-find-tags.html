NAME

pg-find-tags - Search for PG files that contain the specified metadata tags.

SYNOPSIS

 
        pg-find-tags ~/MyLibrary /ww/OtherLibrary --author=Rogawski --edition=1

DESCRIPTION

Recusively searches the paths given for PG files containing all of the specified tags. Output is the path to each matching file. Legal tags are as follows:

Global fields:

        --DESCRIPTION=STRING
        --KEYWORDS=STRING
        --DBsubject=STRING
        --DBchapter=STRING
        --DBsection=STRING
        --Date=STRING
        --Institution=STRING
        --Author=STRING

Text-specific fields:

        --title=STRING
        --edition=STRING
        --author=STRING
        --chapter=STRING
        --section=STRING
        --problem=STRING

If multiple text-specific fields are given, then all must match for a single textbook.

LIMITATIONS

Doesn't support full boolean searches, and it probably should. Can only match on full strings, so you can't match on a single keyword, for example.
