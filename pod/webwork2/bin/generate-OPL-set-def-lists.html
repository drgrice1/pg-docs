NAME

generate-OPL-set-def-list - find all set definition files in the OPL and Contrib

SYNOPSIS

generate-OPL-set-def-list

The environment variable $WEBWORK_ROOT must be set with the location of webwork2, and either the environment variable $PG_ROOT must be set with the location of pg, or pg must be located in the parent directory of the webwork2 location.

DESCRIPTION

This script will find all set definition files in the OpenProblemLibrary and Contrib subdirectories of the webwork-open-problem-library and list them in the files $WEBWORK_ROOT/htdocs/DATA/library-set-defs.json and $WEBWORK_ROOT/htdocs/DATA/contrib-set-defs.json.
