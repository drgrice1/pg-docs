NAME

newpassword - change a users password

SYNOPSIS

 
 newpassword COURSEID USERID NEWPASSWORD

DESCRIPTION

Change the password for a user.

ARGUMENTS
COURSEID

The name of the course.

USERID

The login name of the user you will change the password for.

NEWPASSWORD

The new password.
