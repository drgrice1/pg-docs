NAME

pg-append-text-tags -- Add textbook tags to a PG file.

SYNOPSIS

 
        pg-append-text-tags file1.pg file2.pg --author=Rogawski \
        --title='Calculus: Early Transcendentals' --edition=1 \
        --chapter=3 --section=1 --problem=11,13

DESCRIPTION

This script appends metadata tags for a new textbook to one or more PG files. Tags are given as switches on the command line:

        --title=STRING
        --edition=STRING
        --author=STRING
        --chapter=STRING
        --section=STRING
        --problem=STRING

More than one problem can be specified for --problem by passing a comma-separated list.

LIMITATIONS

Only adds tags for a new textbook, can't rewrite existing tags, can't remove tags.

At some point I will write Tie::PGFile, which will allow direct editing of tags just by modifying a hash, and that will fix all of these problems. :)
