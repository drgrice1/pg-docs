NAME

ww_purge_old_nonces Delete nonce records from Key table for timestamps that are older than ten seconds.

SYNOPSIS

 
 ww_purge_old_nonces  course

DESCRIPTION

Deletes nonce records from the Key table if their timestamps are more than 10 seconds old.

OPTIONS
course

Course for which old nonces should be deleted.
