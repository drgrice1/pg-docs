NAME

pg-perltidy.pl -- Run perltidy on pg problem files.

SYNOPSIS

 
  pg-perltidy.pl [options] file1 file2 ...

DESCRIPTION

Run perltidy on pg problem files.

OPTIONS

This script accepts all of the options that are accepted by perltidy. See the perltidy documentation for details.

Note that if the -pro=file option is not given, then this script will attempt to use the .perltidyrc file in the PG root directory for this option. For this to work the PG_ROOT environment variable must be set, and the .perltidyrc file in the PG root directory must be readable.
