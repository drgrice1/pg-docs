NAME

MathObjects.pl - Macro-based fronted to the MathObjects system.

DESCRIPTION

This file loads Parser.pl which in turn loads Value.pl The purpose of this file is to encourage the use of the name MathObjects instead of Parser (which is not as intuitive for those who don't know the history).

It may later be used for other purposes as well.
