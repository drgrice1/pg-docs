NAME

PGstandard.pl - Load standard PG macro packages.

SYNOPSIS

 
 loadMacros('PGstandard.pl');

DESCRIPTION

PGstandard.pl loads the following macro files:

  • PG.pl

  • PGbasicmacros.pl

  • PGanswermacros.pl

  • PGauxiliaryFunctions.pl

  • customizeLaTeX.pl
