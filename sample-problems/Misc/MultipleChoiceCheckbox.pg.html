Setup

To setup, CheckboxList creates a new object. The format is $checks = CheckboxList([choices, ...], [correct_choices, ...], options); where the correct_choices can either match those in choices or be the indices (starting at 0).

If we nest choices in another arrayref, then the choices will be shown randomly.

Using the option separator and setting to &nbsp;, the HMTL space, results in a horizontal checklist.

Lastly, see the POD for more options.
