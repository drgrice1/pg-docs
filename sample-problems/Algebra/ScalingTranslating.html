Preamble

We must load parserFunction.pl so that we can add a named function to the context.

parserFunction(f => 'sin(e * x) + 5.5 * pi * x^2');

$answer = Formula('f(x - 2) + 1');

Setup

The parserFunction method allows us to add a named function to the context. We can define this function however we want, so we chose a function whose formula the students will not guess, whose domain is all real numbers, and which will have no issues during answer evaluation. Once a named function is added to the context, you can use it like you would any other named function.
